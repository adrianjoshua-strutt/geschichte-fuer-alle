# Content Directory

This directory contains the content files for each location/page.

## File Structure

For each location, you need two files:
- `{location}.txt` - The text content to display
- `{location}.mp3` - The audio file (optional)

## Adding a New Location

1. Create a text file: `{location}.txt` with the content you want to display
2. (Optional) Add an audio file: `{location}.mp3`
3. The location will be accessible at: `https://yourusername.github.io/yourrepo/{location}`

## Example

For a location called "dom":
- Create `dom.txt` with text content
- Add `dom.mp3` with audio content
- Access at: `.../dom`

## Current Locations

- `dom` - Information about the Fuldaer Dom
- `stadtschloss` - Information about the Fuldaer Stadtschloss

## Note about MP3 Files

MP3 files are not included in the repository due to file size. Please add your audio files directly to this directory after cloning the repository.
